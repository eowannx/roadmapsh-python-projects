# Number Guessing Game

A command-line number guessing game, built with Python.

## Requirements

- Python 3.6+
- Terminal

## Setup

This project is part of a larger repository with multiple projects. Clone the whole repo and navigate to this folder:

```bash
git clone https://github.com/eowannx/roadmapsh-python-projects.git
cd number-guessing-game
```

## Usage

```bash
python number_guessing_game.py
```

## How it works

The game picks a random number between 1 and 100. Choose a difficulty level and try to guess it within the allowed attempts. After each guess you get a hint — higher or lower.

| Difficulty | Attempts |
|---|---|
| Easy | 10 |
| Medium | 5 |
| Hard | 3 |
